## Methodology Notes

- **Projected Pick**: From Tier 1 XGBoost regressor trained on FanGraphs stats + conf_strength + conference-adjusted stats
- **Range Band**: ±MAE (Mean Absolute Error) from cross-validation — 108.6 picks for hitters, 111.6 for pitchers
- **Delta (Δ)**: Projected Pick minus Actual Pick. Positive = model thought they'd go later (drafted higher than expected). Negative = model thought they'd go earlier (drafted lower than expected).
- **Status**: ✓ = within MAE band. ⬆ HIGHER = drafted earlier than model's range. ⬇ LOWER = drafted later than model's range.
- **Grade**: Model's composite value grade (elite/high/medium/low) combining draft position + MLB probability projections.
- **College filter**: Excluded all high school picks (HS SR). Included 4-year, JUCO, and graduate students.
- **Player matching**: Name-normalized matching with nickname resolution (Chris→Christopher, etc.) and school-based disambiguation for duplicate names.

### Interpretation

The MAE band is deliberately wide (~3.5 rounds) because the model uses only
publicly available FanGraphs aggregate stats — no scouting grades, no
TrackMan data, no medical information, no signability intel. A ±108 pick
band reflects the inherent noise in predicting a process heavily influenced
by team-specific needs, bonus pool allocation, and subjective evaluation.

The internal draft model (see ) is designed to
narrow this band by adding TrackMan-derived physical metrics (EV90, FB velo,
command precision, etc.) that the public model cannot access.
